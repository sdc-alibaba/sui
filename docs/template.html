开发者文档

引入underscore中的JS模板，underscore地址：undercore template。

JS用法

var compiled = $.template("hellow <%= name %>"); 
compiled({"name": "world"});
  • $.template 方法接收一个字符串模板，返回一个compiled方法，此方法接收一个JSON对象并返回解析后的模板。

模板语法

  • <% xxx %> 编写任意JS代码
  • <%= name %> 输出变量
  • <%- name %> 输出HTML转义的变量
