SUI

SUI 是一套基于bootstrap开发的前端组件库，同时她也是一套设计规范。
通过SUI，可以非常方便的设计和实现精美的页面。

丰富的组件库

  • 响应式栅格
  • icon font 字体图标
  • 按钮，表单，表格，提示
  • 标签页，步骤条，面包屑
  • 对话框，Tooltip，分页器
  • 更多组件敬请期待...

为现代浏览器设计

  • 扁平化设计
  • 响应式布局
  • icon font
  • CSS 特效
  • 兼容chrome,firfox,safari,ie8+等主流浏览器
