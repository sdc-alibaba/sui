全局样式的修改主要包含以下几点

              
    
                
  • 字体：font-family: Arial, "微软雅黑", "宋体", sans-serif; font-size: 12px; line-height: 18px; color: #333333;
    • 
                
  • 链接颜色：修改了默认的链接颜色，默认颜色改为 #06c, hover 状态颜色改为 #004080
    • 
                
  • 移除了body元素的margin
    • 
                
  • 标题：从 h1 至 h5 的 font-size分别是：24px,22px,20px,14px,12px，行高是文字大小的1.5倍，上下margin都是9px
    • 
                
  • 段落：默认段落之间的间距是默认的行高 18px
    • 
                
  • 列表：移除了 ul 和 ol 的默认margin和padding
    • 
              

              

                除此之外，还对 small, strong, blockquote等样式进行了调整。

                对于 form，table 等的样式设置大部分都被封装在了对应的class之下，不会影响全局样式。
