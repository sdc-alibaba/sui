级联选择  Tree

根据给定的数据生成下拉框，并且能够级联选择，并取到相应的数据。

1. select，通过标记自动初始化

  • 示例
  • 代码
  • 文档
  • API

            

            

              data-toggle='tree'通过在元素上加此属性自动初始化，并生成第一个select，必须在初始化之前指定数据源

              data-src="url"  通过ajax或jsonp方式获取数据的url

              data-jsonp="true"  为true表示src的url为垮域请求，会以jsonp的形式发送

              data-val="[x,y,z]"  初始化的时候的初始值，注意xyz的值必须要能在srouce对象中找到

            

            

              $(element).data('tree')，element为data-toggle='tree'的元素，可以拿到当前的值对象，格式为{value : [], option : []}，value代表select的value值，option表示显示到页面的option的文字

              $(element).tree('setValue', [x,y,z])  设置tree的值，参数同初始化的data-val相同

              直接调用suitree方法，并传入参数

              如果不想通过ajax或jsonp的请求，可以直接在source参数后传入json对象，注意格式要正确

              自定义数据的格式：

              
{
  "nodes" : [
    {
      "id" : "id", // value值
      "value" : "value", // 显示的内容
      "parent" : "parent", // TODO ，非必需，暂未支持
      "isleaf" : "false", // 是否是叶结点，如果为true，则不会解析children里面的内容
      "childred" : [...] // 子节点，相同的格式
    },
    {...},
    ...
  ]
}
